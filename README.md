# 🍷 BlockBar Launcher

Офіційний лаунчер сервера **BlockBar**.

Сучасний лаунчер, створений для максимально простого запуску Minecraft із готовим модпаком, автоматичними оновленнями та зручним інтерфейсом.

---

## ✨ Можливості

- 🚀 Автоматичне встановлення лаунчера
- 🔄 Автоматичне оновлення файлів
- 📦 Завантаження лише необхідних файлів
- 🎮 Запуск модпака одним натисканням
- 👤 Підтримка Minecraft-аватара за нікнеймом
- 🌍 Українська та англійська мови
- 📰 Вбудована сторінка новин
- ☕ Сторінка підтримки проєкту
- ⚡ Сучасний інтерфейс на Electron

---

## 📥 Встановлення

1. Відкрийте вкладку **Releases**.
2. Завантажте останню версію **BlockBar Launcher Web Setup**.
3. Запустіть інсталятор.
4. Дочекайтеся завершення встановлення.
5. Введіть свій Minecraft-нікнейм.
6. Натисніть **PLAY**.
